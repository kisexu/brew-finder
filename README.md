# 🍺 Brew Finder

Languages: English | [简体中文](README.zh-CN.md)

A Chrome extension that automatically detects whether the software website you are visiting has a matching Homebrew package and lets you quickly copy the `brew install` command.

## Features & Highlights

- **Automatic discovery** — When you visit a relevant site, the extension automatically checks for packages related to that domain and alerts you through the badge or page overlay.
- **Quick actions** — View basic Homebrew package details and copy install commands with one click. Covers official formulae and casks.
- **Large-site support** — Especially for domains like Google.com that are associated with thousands of packages. Brew Finder helps users find the packages they need while reducing disruption to normal browsing.
- **GitHub matching** — Visiting `github.com/user/repo` can also identify the matching Homebrew package.
- **Privacy by design** — This extension has no network functionality. All package information is cleaned and bundled into the extension while keeping the package size lean.
